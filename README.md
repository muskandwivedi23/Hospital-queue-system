# 🏥 Hospital Queue Management System

A MERN Stack based Hospital Queue Management System that helps hospitals manage patient queues efficiently. It provides separate interfaces for Admin, Staff, Kiosk, and Display screens to improve patient flow and reduce waiting time.

## 🚀 Features

- Patient Token Generation
- Live Queue Display
- Staff Dashboard
- Admin Dashboard
- Kiosk Interface
- Real-Time Queue Management

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- CSS
- Git & GitHub

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/muskandwivedi23/Hospital-queue-system.git
```

Install dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

Run the project

```bash
# Backend

cd backend
npm start

# Frontend
cd ../frontend
npm run dev
```
